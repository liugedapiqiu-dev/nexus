## 🚀 使用方法

### 1. 扫描整个技能目录
```bash
skillguard scan /home/user/.openclaw/workspace/skills/
```

### 2. 检查单个技能文件
```bash
skillguard check /path/to/skill.md
```

### 3. 扫描并输出报告到文件
```bash
skillguard scan /home/user/.openclaw/workspace/skills/ --output report.txt
```

### 4. JSON 格式输出（用于自动化）
```bash
skillguard scan /home/user/.openclaw/workspace/skills/ --format json
```

### 5. 只显示高风险以上的问题
```bash
skillguard scan /home/user/.openclaw/workspace/skills/ --severity high
```

### 6. 详细模式（显示代码样本）
```bash
skillguard scan /home/user/.openclaw/workspace/skills/ --verbose
```

## 🎯 能检测什么

### 🚨 严重（Critical）
- 硬编码的 API 密钥和秘密
- 嵌入的私钥
- Shell 注入模式
- AWS Access Key
- GitHub Token
- OpenAI API Key
- 反向 shell 连接

### ⚠️ 高风险（High）
- 提示注入尝试（"ignore previous instructions"）
- 数据外泄到外部服务
- 危险的 eval() 使用
- Base64 混淆
- 明文密码
- Bearer Token

### ⚡ 中风险（Medium）
- 分隔符注入
- curl 到外部 URL
- chmod 777 过度权限
- 环境变量访问
