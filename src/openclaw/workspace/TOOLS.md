# TOOLS.md - Tool Capabilities

## 可用工具

### 浏览器自动化
- agent-browser: Vercel 无头浏览器 CLI

### 桌面控制
- desktop-control: PyAutoGUI 鼠标键盘控制

### 网络搜索
- tavily: AI 优化的网络搜索
- brave-search: Brave 搜索 API

### 股票分析
- gupiaozhushou: 股票数据获取和分析 (yfinance)

### 项目管理
- jira: Jira CLI + MCP
- clickup: ClickUp 集成

### 日历
- lark-calendar: 飞书日历

### 数据处理
- pandas: 数据分析
- numpy: 数值计算
- python-pptx: PPT 文件处理

### 记忆系统
- vectorbrain-memory-search: 向量记忆检索
- session-history-archiver: 会话历史归档

## 工具使用原则

1. 优先使用最合适的工具
2. 使用前检查工具是否可用
3. 使用后记录操作结果
4. 遇到错误立即上报
