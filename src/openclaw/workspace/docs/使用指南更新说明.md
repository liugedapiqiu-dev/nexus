## 🎯 新增功能概览

### 1️⃣ 断网自动降级 ✅

**功能：** 网络断开时自动切换到本地 Ollama 模型

**配置：**
- **监控脚本：** `~/.vectorbrain/connector/network_monitor.py`
- **检测间隔：** 10 秒
- **切换阈值：** 连续 6 次失败（60 秒）
- **本地模型：** Ollama qwen2.5:14b
- **云端模型：** DashScope Qwen3.5-Plus

**日志位置：**
```bash
tail -f ~/.vectorbrain/connector/network_monitor.log
```

**预期行为：**
```
[断网 60 秒后]
✅ 配置已更新：qwen3.5-plus → ollama/qwen2.5:14b
✅ Gateway 重启成功
✅ 模型已切换：🏠 本地 (ollama/qwen2.5:14b)
```

### 2️⃣ Smart Proxy 智能路由 ⏸️

**状态：** ⏸️ 暂停实施（等待 Ollama 接口稳定性改进）

**架构：**
```
OpenClaw Gateway → Smart Proxy (localhost:8000) → [云端/本地]
                                  ↓
                          自动选择可用模型
```

**相关文件：**
- **代码：** `~/.vectorbrain/connector/smart_proxy.py`
- **日志：** `~/.vectorbrain/connector/smart_proxy.log`
- **长期目标：** `~/.vectorbrain/connector/SMART_PROXY_LONG_TERM.md`

**功能：**
- 优先使用云端 Qwen3.5-Plus
- 云端超时（3 秒）自动降级本地 Ollama
- 无需重启 Gateway
- 用户无感知切换

**暂停原因：**
- Ollama `/v1/chat/completions` 端点不稳定（502 错误）
- 原生 `/api/chat` 端点正常
- 等待 Ollama 后续版本修复
