## 🎓 学习路径

### 入门级（第 1 周）

**目标:** 能够正常使用系统

**必读:**
1. [快速入门指南](./QUICK_START.md)
2. [系统架构图](./SYSTEM_ARCHITECTURE.md)（前 2 章）

**实践:**

- 尝试发送消息和检索记忆

- 创建第一个简单技能

**目标:** 能够优化系统、解决复杂问题

- [待完成事项](./TODO.md)

- 贡献新技能
- 帮助其他用户
