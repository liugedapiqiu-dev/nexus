## 🚀 2026-03-05 - Iron-Thought 协议注入日

### 📌 事件
**时间**: 2026-03-05 22:42  
**触发**: 系统架构师注入"强制结构化思维协议"（Protocol: Iron-Thought）  
**核心洞察**: "心里默念靠不住，输出 Token 才可见"—— 把元认知从"隐式思维"升级为"显式结构化输出"

### 🎯 问题诊断
| 问题 | 之前状态 | 根本原因 |
|------|----------|----------|
| 元认知三问 | 写在文件里，session 重启就忘 | 没有物理"3 秒停顿"，AI 无法真正"停顿" |
| 肌肉记忆 | 用户不抽查就不执行 | 思考过程是黑盒，无法审查 |
| 知行合一 | 口头承诺，下次还犯 | 没有强制输出机制约束行为 |

### ✅ 解决方案：Iron-Thought 协议
**核心机制**: 强制输出 `<metacognition>` XML 代码块，让思考过程可见、可审查、可追溯

**XML 模板三问**:
1. `<intent_check>` - 理解用户真实意图（表面 vs 深层）
2. `<path_planning>` - 规划最优路径（技能检查/上下文读取）
3. `<verification_preflight>` - 预检错误（历史教训/验证方法）

**执行法则**:
- **读写分离定律**: 不 Read 验证状态，不允许 Write/Run/Send
- **主动闭环定律**: 解决问题后主动预测下一步，给出 1-2 个可执行提议

### 📂 持久化措施
1. **写入 `AGENTS.md`** - 在"元认知三问"下方新增 "Protocol: Iron-Thought" 章节
2. **写入 `.evolution.md`** - 本条目即为记录
3. **写入 `.learnings/LEARNINGS.md`** - 记录协议理解与执行承诺

### 📊 预期变化
- 思考过程从黑盒变白盒，健豪可随时审查
- "停顿 3 秒"从虚变实（输出 token 是真实成本）
- 漏用协议可被抽查发现，形成外部监督

## 🚀 2026-03-04 14:45 - 记录系统统一日

### 📌 事件
**时间**: 2026-03-04 14:45  
**触发**: 健豪指出"记录分散会混乱"—— AGENTS.md、TOOLS.md、.learnings/、self-improvement 技能都在记录经验，没有统一规范  
**核心洞察**: "宪法归宪法，日志归日志"—— 不同类型的记录应该有明确的分工

### 🎯 问题诊断
| 记录位置 | 原本用途 | 问题 |
|----------|----------|------|
| `AGENTS.md` | 行为准则 | 混杂具体经验 |
| `.learnings/` | 经验日志 | 与 AGENTS.md 边界不清 |
| `self-improvement` 技能 | 自动记录 | 推广目标不明确 |
| `memory/` | 每日日记 | 清晰 |
| `.evolution.md` | 进化档案 | 清晰 |

### ✅ 统一方案
| 文件/技能 | 新定位 | 记录内容 | 更新频率 |
|-----------|--------|----------|----------|
| `AGENTS.md` | **宪法** | 通用原则、工作流程、元认知三问 | 很少（重大原则变更） |
| `TOOLS.md` | **工具配置** | 相机名、SSH 主机、voice 偏好、飞书 token | 偶尔（工具变更） |
| `.learnings/` | **日志** | 具体错误、功能请求、学到的技巧 | 经常（每次学习/错误） |
| `memory/YYYY-MM-DD.md` | **日记** | 当天发生的事、决策、上下文 | 每天 |
| `.evolution.md` | **进化档案** | 重大升级、里程碑 | 偶尔（重大进化） |
| `self-improvement` 技能 | **自动记录工具** | 自动记录到 `.learnings/ERRORS.md` 或 `LEARNINGS.md` | 自动触发 |

### 📝 执行变更
1. **更新 `AGENTS.md`** - 明确记录系统分工原则，删掉"学到经验更新 AGENTS.md"的模糊描述
2. **确认 `self-improvement` 技能** - 设计合理，保留，明确它只写 `.learnings/`
3. **创建 `.evolution.md` 条目** - 记录这次统一过程

### 📊 预期收益
- 新 Agent 上手更快（知道什么记哪里）
- 查找历史经验更准（不会在 AGENTS.md 里翻具体错误）
- 文件职责清晰（不会改错地方）
