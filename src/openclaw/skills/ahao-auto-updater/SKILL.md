# [YOUR_AI_NAME]文件变更检测器 (Ahao Auto-Updater)

自动检测关键文件变化并记录到变更日志

## 触发意图

当用户提到以下需求时调用此技能：

- 检测文件变化
- 检查配置变更
- 更新变更日志
- 文件完整性检查
- 技能变更监控
- 文件变化
- 配置变更
- 变更检测
- 更新日志
- 文件监控

## 核心功能

| 功能 | 说明 |
|------|------|
| **文件监控** | 监控关键目录和文件的变化 |
| **哈希比对** | 使用 SHA256 哈希值比对文件内容 |
| **变更日志更新** | 自动更新 CHANGELOG.md |
| **并发保护** | 文件锁机制防止并发冲突 |
| **超时保护** | 10 秒超时保护 |
| **磁盘检查** | 定期清理和磁盘空间检查 |

## 监控范围

### 监控目录

- `~/.openclaw/skills/` - 技能目录
- `~/.openclaw/hooks/` - Hook 配置
- `~/.openclaw/workspace/docs/` - 文档目录
- `~/.vectorbrain/connector/` - 连接器
- `~/.vectorbrain/src/` - 源码

### 文件类型

监控的文件模式：`*.py`, `*.json`, `*.md`, `*.sh`, `*.yaml`, `*.yml`, `*.toml`

排除目录：`node_modules`, `__pycache__`, `.git`, `venv`

## 输出格式

变更日志包含以下部分：

1. **变更摘要** - 总体变化概述
2. **新增文件** - 新创建的文件列表
3. **修改文件** - 内容变更的文件
4. **删除文件** - 被删除的文件
5. **哈希记录更新** - 新的哈希记录

## 配置文件

- 哈希记录文件：`~/.openclaw/workspace/.file_hashes.json`
- 变更日志文件：`~/.openclaw/workspace/CHANGELOG.md`
