# 知识去重 (Knowledge Dedup)

知识记忆去重机制 - 检测 VectorBrain 中的重复知识记录，提供合并建议

## 触发意图

当用户提到以下需求时调用此技能：

- 知识去重
- 检测重复
- 合并知识
- 清理记忆
- 优化存储

## 自动触发

- **频率**：每月自动运行一次
- **手动运行**：`python3 knowledge_dedup.py --force`

## 核心功能

| 功能 | 说明 |
|------|------|
| **相似度检测** | 基于文本相似度检测重复（阈值 85%） |
| **类别分组** | 按类别分组比较，提高效率 |
| **合并建议** | 提供智能合并建议（保留更新的） |
| **置信度评分** | 置信度评分（high/medium） |
| **生成报告** | 生成详细去重报告 |
| **保存到 VectorBrain** | 报告保存到 VectorBrain 知识记忆 |

## 去重配置

| 参数 | 值 |
|------|-----|
| **相似度阈值** | 0.85 (85%) |
| **最大记录数** | 500 |
| **高置信度** | 0.95 (95%) |
| **中置信度** | 0.85 (85%) |

## 置信度等级

| 等级 | 阈值 | 说明 |
|------|------|------|
| **High** | 95% | 高度确信为重复 |
| **Medium** | 85% | 可能为重复 |

## 输出报告

去重报告包含：

1. **重复记录列表** - 疑似重复的记录对
2. **合并建议** - 建议保留哪条记录
3. **置信度评分** - 每条建议的置信度
4. **统计摘要** - 总体去重分析结果

## 使用示例

```
检查重复知识
知识去重分析
合并重复记录
优化知识存储
```

## 注意事项

1. **自动触发**：每月运行一次
2. **安全操作**：只分析，不自动删除
3. **合并建议**：提供详细合并建议
4. **报告保存**：所有报告保存到 VectorBrain
5. **手动运行**：`python3 knowledge_dedup.py --force`
