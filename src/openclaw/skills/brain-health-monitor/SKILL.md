# 大脑健康监控 (Brain Health Monitor)

大脑健康度自动监控 - 定期检查数据库、技能配置、记忆提炼效率，自动汇报健康状态

## 触发意图

当用户提到以下需求时调用此技能：

- 大脑健康检查
- 系统状态检查
- 健康度监控
- 技能配置检查
- 数据库状态
- 记忆系统检查

## 自动触发

- **空闲触发**：用户空闲超过 2 小时自动触发
- **随机检查**：每天随机时间点检查（间隔至少 12 小时）

## 核心功能

| 功能 | 说明 |
|------|------|
| **数据库检查** | 检查所有 VectorBrain 数据库完整性 |
| **技能配置检查** | 检查技能配置完整性（skill.json + SKILL.md） |
| **记忆提炼检查** | 检查记忆提炼效率 |
| **文件清理检查** | 检查是否需要清理备份文件和旧版本 |
| **健康评分** | 计算 0-100 的健康度分数 |
| **自动生成报告** | 自动生成报告并保存 |

## 检查项目

### 数据库检查

- 知识记忆数据库
- 情景记忆数据库
- 反思数据库
- 任务队列数据库
- 目标系统数据库

### 技能配置检查

- skill.json 存在性
- SKILL.md 存在性
- 配置完整性

### 记忆提炼检查

- 提炼效率
- 去重状态
- 知识质量

## 健康度评分

| 分数范围 | 状态 | 建议 |
|----------|------|------|
| 90-100 | ✅ 优秀 | 保持现状 |
| 75-89 | ✅ 良好 | 关注改进点 |
| 60-74 | ⚠️ 一般 | 建议处理 |
| <60 | ❌ 需改进 | 立即处理 |

## 输出报告

- **报告路径**：`~/.openclaw/workspace/memory/brain_health_report.md`
- **状态文件**：`~/.openclaw/workspace/memory/brain_health_state.json`

## 使用示例

```
检查大脑健康状态
系统健康度如何
技能配置完整吗
数据库是否正常
```

## 注意事项

1. **自动触发**：用户空闲>2 小时 或 随机检查（12 小时间隔）
2. **健康度<85**：会建议立即处理
3. **报告保存**：自动保存到 `brain_health_report.md`
4. **状态记录**：检查状态记录在 `brain_health_state.json`
