# 记忆提炼引擎 (Memory Extraction Engine)

记忆自动提炼引擎 - 分析情景记忆，识别高价值对话，自动提炼为知识记忆

## 触发意图

当用户提到以下需求时调用此技能：

- 提炼记忆
- 分析情景记忆
- 提取知识
- 记忆整理
- 知识沉淀

## 自动触发

- **每天自动**：每天自动运行 1 次
- **空闲触发**：用户空闲 4 小时以上时触发
- **最小间隔**：24 小时

## 核心功能

| 功能 | 说明 |
|------|------|
| **分析情景记忆** | 分析最近 24 小时的情景记忆 |
| **内容分类** | 识别架构决策、工作流程、技能配置、经验教训等类别 |
| **提取知识** | 自动提炼高价值内容为知识记忆 |
| **标记已提炼** | 标记已提炼的情景记忆，避免重复 |
| **生成报告** | 生成提炼报告并保存到 VectorBrain |

## 提炼分类

| 类别 | 说明 |
|------|------|
| **architecture_decision** | 架构决策、规范制定、标准定义 |
| **workflow** | 工作流程、操作方法、执行步骤 |
| **skill_configuration** | 技能配置、Function Calling 相关 |
| **experience** | 经验教训、问题解决、错误修复 |
| **system_standard** | 系统标准、必定执行的规范 |

## 提炼流程

```
1. 读取最近 24 小时情景记忆
    ↓
2. 分析对话内容
    ↓
3. 识别高价值信息
    ↓
4. 分类并提炼
    ↓
5. 保存到知识记忆
    ↓
6. 标记已提炼
    ↓
7. 生成报告
```

## 输出报告

- **报告路径**：`~/.openclaw/workspace/memory/extraction_report.md`
- **报告内容**：提炼摘要、分类统计、新增知识列表

## 使用示例

```
提炼今天的记忆
分析最近的经验
提取有价值的对话
整理情景记忆
```

## 注意事项

1. **自动触发**：每天运行 1 次或空闲时触发
2. **避免重复**：已提炼的记忆会被标记
3. **质量保障**：建议人工审核提炼结果
4. **数据保存**：所有数据保存到 VectorBrain 向量数据库
