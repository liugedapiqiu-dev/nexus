# 自动反思引擎 (Auto Reflection Engine)

任务完成后自动反思系统 - 分析已完成任务，提取经验教训，自动记录到 VectorBrain

## 触发意图

当用户提到以下需求时调用此技能：

- 任务反思
- 经验总结
- 教训提取
- 任务复盘
- 自动反思

## 自动触发

- **任务完成**：任务完成后自动触发
- **定期检查**：每 12 小时检查一次
- **空闲触发**：用户空闲 6 小时以上时触发

## 核心功能

| 功能 | 说明 |
|------|------|
| **分析完成任务** | 分析最近 24 小时完成的任务 |
| **提取经验教训** | 提取成功经验和失败教训 |
| **识别模式** | 识别有效的工作模式 |
| **生成行动建议** | 生成后续行动建议 |
| **保存到 VectorBrain** | 保存到知识记忆，支持向量检索 |
| **标记已反思** | 标记已反思的任务，避免重复 |

## 任务类型分类

| 类型 | 说明 |
|------|------|
| **skill_configuration** | 技能配置相关任务 |
| **memory_operation** | 记忆系统操作任务 |
| **health_check** | 健康检查任务 |
| **cleanup** | 清理整理任务 |
| **system_upgrade** | 系统升级任务 |
| **experience** | 其他经验类任务 |

## 反思模板

```markdown
## 成功因素分析
- ...

## 失败原因分析
- ...

## 发现的工作模式
- ...

## 后续行动建议
- ...
```

## 使用示例

```
反思今天完成的任务
总结项目经验
提取失败教训
分析任务模式
```

## 注意事项

1. **自动触发**：任务完成后或定期触发
2. **避免重复**：已反思的任务会被标记
3. **质量保障**：失败任务会标记需要人工审核
4. **数据保存**：所有数据保存到 VectorBrain 向量数据库
